# Pepa Pig - Salta Charcos 🐷

Juego web para niños basado en Peppa Pig. PWA instalable, funciona offline.

## Cómo jugar

**Modo Charcos:** Saltá por encima de los charcos de barro. La velocidad aumenta con el tiempo.

**Modo Lodo:** Aterrizá *dentro* de los charcos. Tenés 3 corazones.

Toca la pantalla o presioná **Espacio** para saltar.

## Personajes

- 🩷 **Pepa** — vestido rojo
- 💙 **George** — remera azul

## Tecnología

HTML + Canvas. Sin dependencias. PWA con service worker para uso offline.

## Deploy

```bash
npx vercel --prod
```
